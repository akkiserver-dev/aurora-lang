# Aurora プログラミング言語

Auroraの公式ドキュメントへようこそ。Auroraは、安全性、パフォーマンス、そして開発者体験を重視して設計された汎用プログラミング言語です。

## 言語リファレンス

- [基本構文](lang/basics.md): 構文、キーワード、識別子。
- [制御フロー](lang/control_flow.md): If、match、およびループ。
- [関数](lang/functions.md): 関数定義、ラムダ、拡張関数。
- [オブジェクト指向](lang/oop.md): クラス、継承、トレイト。
- [並行処理](lang/concurrency.md): スレッドと `async`/`await`。
- [パッケージシステム](lang/packages.md): 名前空間と可視性。
- [エラーハンドリング](lang/error_handling.md): Result型と try-catch。

## 型システム

- [プリミティブ型](type/primitives.md): 数値型、文字列、真偽値。
- [Null安全性](type/null_safety.md): Null許容型とスマートキャスト。
- [ジェネリクス](type/generics.md): 型安全な再利用可能コード。
- [レコードと列挙型](type/records.md): データオブジェクトと列挙型。
- [コレクション](type/collections.md): 配列、リスト、セット、マップ。
- [標準ライブラリ](type/stdlib.md): IoやConcurrentなどのコアモジュール。
